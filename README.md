# Slice Therapy

A Unity-based wrist rehabilitation game designed to help users improve hand and wrist mobility through interactive vegetable-slicing exercises.

## Overview

Slice Therapy is a serious game developed as part of a Unity Developer technical assignment. The game focuses on wrist rehabilitation by encouraging controlled hand movements through a simple kitchen-themed activity.

Players slice vegetables and move them into a vessel, helping improve coordination, precision, and wrist range of motion in an engaging and motivating environment.

## Features

- Wrist rehabilitation focused gameplay
- Vegetable slicing mechanics
- Object transfer and placement tasks
- Simple and intuitive controls
- Android APK build
- WebGL browser build
- Developed using Unity and C#

## Gameplay

The player performs controlled wrist-like movements to:

1. Slice vegetables on a cutting board.
2. Move the sliced vegetables into a vessel.
3. Repeat the process to complete rehabilitation-oriented tasks.

The game is designed to simulate repetitive therapeutic movements while maintaining an enjoyable gameplay experience.

## Controls

### Keyboard

| Key | Action |
|------|--------|
| Up Arrow | Move wrist/tool upward |
| Down Arrow | Move wrist/tool downward |
| Side Movement Key | Move sliced vegetables into the vessel |

### Mobile

- Touch controls are supported for Android devices.

## Assignment Objective

This project was created to address the following rehabilitation scenario:

- A patient suffers from wrist movement limitations.
- The patient experiences difficulty performing kitchen-related tasks.
- The patient requires a way to practice and observe improvement through interactive exercises.

The game uses vegetable slicing activities to simulate practical wrist movements in a controlled environment.
